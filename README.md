# 팀 짜줍니다

구성원 파일을 업로드하면 랜덤으로 팀을 배정해주는 웹 도구입니다.
순수 HTML/CSS/JS로 만들어졌으며 서버 없이 브라우저에서만 동작합니다.

## 기능

- txt 파일에서 구성원 이름 읽기
- **팀당 인원** 또는 **팀 개수** 기준 선택 가능
  - 팀당 인원 기준 → 슬라이싱 방식
  - 팀 개수 기준 → 라운드로빈 방식으로 균등 배분
- 총 인원 및 팀 수 표시
- 이름 카드 드래그로 팀 간 이동
- 바뀐 인원 구성 그대로 다시 섞기
- 팀 결과 이미지 클립보드 복사 (html2canvas)
  - 브라우저 정책상 막히면 PNG 다운로드로 자동 대체

## 실행 방법

### 로컬

`start.command` 더블클릭 (Mac) — Python 로컬 서버 실행 후 브라우저 자동 오픈

또는 VS Code에서 `Live Server` 확장으로 실행

### 배포

Vercel로 배포 중 — GitHub 푸시 시 자동 재배포

## txt 파일 형식

한 줄에 이름 하나씩 작성합니다.

```txt
김민지
이서연
박지훈
최유진
```

UTF-8 인코딩을 권장합니다.
