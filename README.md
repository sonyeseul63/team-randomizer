# 랜덤 팀 배정기 MVP

순수 HTML/CSS/JS로 만든 랜덤 팀 배정기입니다.

## 기능

- txt 파일에서 구성원 이름 읽기
- 총 인원 표시
- 팀당 인원 기준으로 랜덤 팀 생성
- 마지막 줄은 기준 인원보다 적으면 `깍두기`로 표시
- 이름 박스를 드래그해서 다른 팀으로 이동
- 또 섞기
- 팀 결과를 이미지로 클립보드 복사
  - 브라우저 정책상 막히면 PNG 다운로드로 대체

## VS Code 실행 방법

1. VS Code에서 이 폴더 열기
2. Extensions에서 `Live Server` 설치
3. `index.html` 우클릭
4. `Open with Live Server` 클릭
5. 브라우저에서 `sample-members.txt`를 넣고 테스트

## txt 파일 형식

한 줄에 이름 하나씩 작성합니다.

```txt
김민지
이서연
박지훈
최유진
```

macOS txt 파일도 가능합니다. 한글 이름이면 UTF-8 저장을 권장합니다.
